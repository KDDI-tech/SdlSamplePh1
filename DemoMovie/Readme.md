# SDL_Demo_Phase1
------------------
* 車内の情報（車輌情報）をAndroidアプリ側に通知し、Androidアプリ側で受け取ったデータを事前に実装した判断ロジックに基づいてSDL機器側に動的に内容を表示する方法を提示する 
  

## SDL_Demo_Phase1.mp4
------------------

| 時間 | 説明 |
| ---- | ---- |
|開始〜|スマホをHU(ヘッドユニット)に接続します|
|0:05〜|HUに接続されたアプリのアイコンが表示されます|
|0:07〜|アプリを起動するとスマホ側がロックスクリーンに切り替わります|
|0:15〜|タイヤ空気圧のデモ<br>ステータスがLOWの場合、空気圧が低いとして通知を行います<br>ステータスがFAULT/ALERの場合、空気圧の検知に異常が発生したとして通知を行います|
|0:45〜|ヘッドライトのデモ<br>周囲が暗いときに、ヘッドライトが無灯火の場合、点灯するように通知します<br>周囲が明るいときに、ヘッドライトが点灯しているの場合、消灯するように通知します|
|1:05〜|燃料のデモ<br>特定の閾値(デモでは50%以下10%毎に閾値を設定)に燃料が減ると、通知を行います|
|1:33〜|HU(ヘッドユニット)との接続を終了します|
